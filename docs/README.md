# CoastCams Website

This directory contains the GitHub Pages website for CoastCams.

## Structure

- `index.html` - Main webpage
- `css/styles.css` - Styling with white theme and blue/red colors
- `js/scripts.js` - Interactive features and functionality

## Viewing the Site

Once GitHub Pages is enabled, the site will be available at:
https://nuytssiegmund.github.io/CoastCams/

## Local Development

To view the site locally, simply open `index.html` in a web browser.

## Updating Content

- **Statistics**: Update the numbers in the hero section of `index.html`
- **Publications**: Add new publications to the Publications section
- **News**: Add new articles to the News section
- **Case Studies**: Add new case studies with images to the Case Studies section

## GitHub Pages Setup

1. Go to repository Settings
2. Navigate to Pages section
3. Set Source to "Deploy from a branch"
4. Select branch: `main` (or your default branch)
5. Set folder to `/docs`
6. Click Save
